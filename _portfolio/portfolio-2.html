Zero-Shot Learning using RoBERTa

Fall 2024

This research project focused on improving zero-shot classification performance through advanced prompt engineering and optimization techniques.

Key Achievements:

  • Significant Accuracy Improvement: Increased classification accuracy from 48.5% to 82.13% - a remarkable 33.63 percentage point improvement
  • Iterative Optimization: Developed a systematic approach to optimize label prompts using state-of-the-art generative language models
  • Multi-Model Analysis: Conducted comprehensive evaluation across multiple large language models including Gemma2-9B, Qwen2.5-32B, and Nemotron-70B
  • Performance Benchmarking: Achieved best results with Nemotron-70B, demonstrating the importance of model scale in prompt optimization

Technical Approach:

  • Base Model: RoBERTa for foundational text understanding
  • Prompt Engineering: Systematic optimization of label prompts for zero-shot scenarios
  • Generative Models: Leveraged multiple LLMs for prompt refinement and optimization
  • Evaluation Framework: Comprehensive testing and validation across different model configurations

Impact:

This work demonstrates the potential of combining traditional transformer models with modern generative AI for significant performance improvements in classification tasks, particularly valuable for scenarios with limited labeled data.
